# SnakeSolo

SnakeSolo es un juego Snake independiente para navegador, extraído de un proyecto más grande y rehecho como proyecto autónomo con ayuda de ChatGPT y Codex.

## Qué se hizo
- Se separó el juego Snake de una app más grande.
- Se eliminó la dependencia de FastAPI y de otras partes de la app original.
- Se creó una versión independiente con:
  - `index.html`
  - `styles.css`
  - `snake-game.mjs`
  - `README.md`

## Qué se tocó
- Estructura HTML del juego
- Estilos CSS del Snake
- Lógica JS del juego
- Estado inicial (`Ready`)
- Botones `Start`, `Pause`, `Restart`
- Movimiento, score y `Game Over`

## Tiempo de trabajo
Proyecto rehecho paso a paso con iteraciones, pruebas locales y correcciones hasta dejarlo jugable como versión independiente.
